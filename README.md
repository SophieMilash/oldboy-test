# OldBoy test

Тестовое задание
* На мобильной и планшетной версиях меню выполнено виде бургера (с использованием JS)
* Полностью кастомизированная форма (согласно макету)
* При попытке отправить форму, в которой поля для ввода телефона или почты заполнены данными в неподходящем формате, появляется встроенное в браузер сообщение об ошибке

**В проекте используются:**
* HTML5
* SASS
* JavaScript
* БЭМ
* Инструмент автоматизации Gulp
* Адаптивная верстка (Desktop First)
* CSS Grid Layout
* CSS Flexbox Layout
